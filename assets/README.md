## 🖼️ Adding Your Images

### Profile Picture (profile.jpg)
- **Recommended Size:** 500x500px or larger (square)
- **Format:** JPG or PNG
- **Tips:** 
  - Use a professional photo
  - Ensure good lighting
  - Plain or simple background works best
  - Center your face in the image

### Project Images (project-images/)
- **Recommended Size:** 1200x600px or 1920x1080px
- **Format:** JPG or PNG
- **Tips:**
  - Use screenshots of your actual projects
  - Show the main UI or key features
  - Ensure text is readable
  - Use consistent aspect ratios for all project images

### Certificate Images (certificates/)
- **Recommended Size:** 1200x900px (A4 aspect ratio)
- **Format:** JPG or PNG
- **Tips:**
  - Scan or photograph your certificates
  - Ensure the text is clear and readable
  - Crop excess white space
  - Maintain aspect ratio when resizing

## 🎨 Image Optimization Tips

1. **Compress Images:** Use tools like TinyPNG or ImageOptim to reduce file size without losing quality
2. **Keep File Sizes Small:** Aim for under 500KB per image for faster loading
3. **Use Consistent Naming:** Use lowercase and hyphens (e.g., `my-project.jpg`)
4. **Modern Formats:** Consider using WebP format for better compression

## 🔧 Tools for Image Editing

- **Free Online Tools:**
  - [Canva](https://www.canva.com/) - Create graphics and edit images
  - [Remove.bg](https://www.remove.bg/) - Remove image backgrounds
  - [TinyPNG](https://tinypng.com/) - Compress images

- **Desktop Tools:**
  - GIMP (Free, cross-platform)
  - Paint.NET (Free, Windows)
  - Photoshop (Paid)

## 📝 Placeholder Images

The current placeholder images are SVG files. Replace them with your actual images by:

1. Saving your image with the exact same filename
2. Placing it in this folder (or the appropriate subfolder)
3. The website will automatically use your new image

## ⚠️ Important Notes

- Do NOT delete the placeholder images until you have replacements ready
- Keep backup copies of your original high-resolution images
- Respect copyright - only use images you have the right to use
- Test your images on different devices to ensure they look good
